Quick Start

SmartPad is a text-first computation pad. Type naturally and use => to ask for an immediate result.

hours = 38
rate = $65
weekly_pay = hours * rate => $2,470.00

Variables and dependencies

Define values once, then reuse them. Updating an upstream variable recalculates dependent results.

radius = 5 m
height = 10 m
volume = PI * radius^2 * height => 785.40 m^3

Live result behavior

  • Use => to display a result in-line.
  • Incomplete comparators like trailing = are intentionally suppressed to avoid parser leakage.
  • Assignments with supported phrase forms (like percentage forms) can still show previews.

Units and conversion

distance = 12 km
distance to mi => 7.46 mi

force = 100 N
