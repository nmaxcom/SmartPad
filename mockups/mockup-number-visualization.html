SmartPad mockups

Ways to visualize large numbers

Each card shows a different way to present scale while keeping the raw value readable.

Grouped digits Readable
light speed => 300 000 000 m/s
Insert separators (spaces or underscores) to show magnitude at a glance.
Compact suffix Fast scan
light speed => 300,000,000 m/s 300M
Keep full precision plus a short badge (K, M, B) for quick scanning.
Hover scale hint On demand
light speed => 300000000 m/s
Keep line clean, show scientific or verbal scale on hover.
Magnitude pill Context
light speed => 300000000 m/s 1e8
Small chip communicates order of magnitude (1e8, 8 digits).
Microbar scale Visual
light speed => 300000000 m/s
A logarithmic bar gives a quick sense of scale while preserving the exact value.
Hybrid (suffix + tooltip) Power user
light speed => 300M m/s
Show compact value by default, reveal exact digits on hover.
Mantissa + exponent chips Math-native
light speed => 3.00 x 10^8 m/s
Break the number into mantissa + exponent with a focused exponent chip.
Stacked detail line Clarity
light speed => 300,000,000 m/s
Scientific: 3.00e+8
Keep the full number but add a quieter scientific readout below.
Digit group emphasis Chunked
light speed => 300 000 000 m/s
Alternate group emphasis so your eye snaps to the magnitude blocks.
Log-scale ruler Scale
light speed => 300000000 m/s
10^4
10^8
A compact ruler reinforces where the value sits on a log scale.
Reference anchor Comparative
light speed => 300000000 m/s
≈ 7.5× Earth circumference per second (Earth ≈ 40,075 km)
Tiny value focus Micro
particle size => 0.00000042 m
Dim the leading zeros, spotlight the first significant digits.
Prefix badge SI
particle size => 420 nm m
Show a scaled SI prefix badge while keeping the base unit visible.
Scientific hint line Dual
particle size => 0.00000042 m
= 4.2e-7 m
Plain decimal + scientific line for quick verification.
Tiny scale microbar Log
particle size => 4.2e-7 m
Tiny values get a short bar to hint at their order.
Inverse scale label Context
particle size => 4.2e-7 m 1 in 2.4M
Frame tiny values as “one in N” to make scale tangible.
