# Getting Started With SmartPad

SmartPad feels like notes, but every line can become a live computation.

## 60-second first win

```smartpad
hours = 38
rate = $95/hour
weekly pay = hours * rate => $3,610

fx = weekly pay in EUR => EUR 3,340
```
