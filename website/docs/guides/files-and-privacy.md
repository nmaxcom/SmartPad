# Files & Privacy

SmartPad is local-first. Your sheets are saved in your browser profile, and exports are plain Markdown you can keep elsewhere.

## What stays local

- Sheets are saved in browser storage on your machine.
- SmartPad does not send sheet text, calculations, variables, or imported files to a SmartPad backend.
- Browser profile resets, private browsing, device moves, or cleanup tools can remove local browser data.

## Back up important work

- Download one sheet as `.md` when you want a readable copy.
- Use `Download All` before switching browsers, devices, or profiles.
- Remove sensitive data before sharing screenshots, videos, or issue examples.

## Current limits

- There are no accounts, cloud sync, team workspaces, or collaboration yet.
- Desktop builds are still a beta path until signed installers exist.
- Currency conversions may depend on external FX providers or cached rates.

## If something feels wrong

Use [Troubleshooting](../troubleshooting) for calculation and syntax issues, or [Support](../support) when storage, import, export, or settings behavior looks wrong.
